# The Humility Journey

A web-based self-reflection and growth tool designed to help you cultivate humility through a 30-day challenge, interactive quiz, and daily journaling prompts. No accounts, no tracking, and no data collection—your journey is entirely private.

## Features

- **Humility Assessment Quiz**: Evaluate your humility across multiple dimensions (Self-Awareness, Others-Focused, Growth, Presence, Intellectual Humility, Self-Worth, Equality, Kindness, Resilience).
- **30-Day Challenge**: Daily themed prompts and micro-practices to build humility, with journaling guidance for each day.
- **Score Breakdown**: Personalized feedback and score levels to help you understand your strengths and areas for growth.
- **Beautiful, Minimal UI**: Elegant design for distraction-free reflection.
- **No Accounts or Tracking**: 100% privacy—no data is stored or sent anywhere.

## Getting Started

1. **Clone the repository**
   ```bash
   git clone <repo-url>
   ```
2. **Install dependencies** (if using npm/yarn for React/TypeScript)
   ```bash
   npm install
   # or
   yarn install
   ```
3. **Run locally**
   - If using a React setup: `npm start` or `yarn start`.
   - Or simply open `index.html` in your browser for a static experience.

## Project Structure

- index.html: Main HTML file, includes styles and app container.
- index.tsx: Main React component, quiz logic, challenge prompts, and UI.

## How It Works

- **Quiz**: Answer 15 questions to assess your humility. Scores are calculated and categorized into four levels.
- **Challenge**: Each day offers a unique theme, prompt, and micro-practice. Reflect in your own notepad or journal.
- **Results**: Receive feedback and suggestions based on your quiz score.

## Customization

- You can expand the quiz, add new challenge days, or modify prompts to suit your needs.
- All logic and UI are contained in index.tsx for easy editing.
